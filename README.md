# haut-acm-bug
## 简介
> 本项目用于[HAUT OnlineJudge System](http://218.28.220.249:50015/JudgeOnline)的`bug`汇总
## `issue`提交步骤  
1. 点击本页面上方`Issues`  
2. 点击新页面右上方绿色`New issue`  
3. 在新页面中部输入反馈,点击右下角绿色`Submit new issue`
